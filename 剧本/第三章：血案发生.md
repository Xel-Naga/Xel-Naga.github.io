## 二、分场景剧本

### **场景一：清晨惊魂（6:30-7:15）**

**【场景描述】**
清晨6:30，天色微明。暴雪稍缓，但天空依然阴沉。悬云观被厚厚积雪覆盖，屋檐垂下冰凌。藏经阁位于道观西北角，是一座独立的二层木结构建筑，青瓦飞檐，门窗紧闭。

**【动作指示】**
1. **明心道士清扫**：明心穿着厚棉袍，手持竹扫帚清扫庭院积雪。他动作机械，口中呼出白气。
2. **发现异常**：扫到藏经阁门前时，明心注意到门下缝隙渗出暗红色粘稠液体。液体已部分凝固，呈暗红色，在白雪衬托下格外刺眼。
3. **紧急反应**：明心蹲下用手指轻触液体，凑近闻了闻，脸色骤变。他扔下扫帚，踉跄跑向主殿。
4. **撞倒香炉**：明心冲进主殿时撞倒青铜香炉，金属撞击声在寂静清晨格外刺耳。
5. **惊醒众人**：香炉倒地声惊醒所有人。林墨和苏晓雨从客房冲出，陈青山从厨房跑来，其他人陆续出现。

**【对话脚本】**
```
明心：（气喘吁吁，声音颤抖）血...藏经阁...门下...有血！

玄真道长：（快步上前，抓住明心肩膀）冷静！说清楚！

明心：藏经阁门下渗出暗红色液体，像是...凝固的血...

林墨：（与苏晓雨对视一眼）道长，藏经阁钥匙在谁那里？

玄真道长：（从腰间取下一串古铜钥匙）在我这里。但昨夜10点我已锁门。

陈青山：先去看看情况。
```

**【破门而入】**
1. **集合前往**：众人跟随玄真道长走向藏经阁。雪地上留下一串凌乱脚印。
2. **钥匙尝试**：玄真道长用钥匙开锁，锁芯转动正常，但门推不开——内部有门闩。
3. **撞门准备**：林墨和陈青山拆下旁边厢房的门闩作为撞木。
4. **三次撞击**：
   - 第一次：门闩松动，发出“嘎吱”声
   - 第二次：门轴断裂声，门向内倾斜
   - 第三次：门向内倒塌，灰尘扬起
5. **视觉冲击**：门倒瞬间，清晨微光射入，照亮悬浮的尘埃颗粒。赵老板的尸体在光束中呈现——标准打坐姿势，胸口插着古铜剑，周围用朱砂绘制完整八卦阵。

**【苏晓雨反应】**
苏晓雨第一个看到尸体，尖叫后退撞到书架。林墨迅速扶住她，同时目光扫视现场。

**【周女士异常】**
周女士看到尸体后，瞳孔放大，喃喃自语：“又是这样...百年轮回...”随后身体一软，陈青山及时扶住她。

## 三、机关原理详解（墨理专业设计）

### **1. 重力平衡密室机关**

**【物理原理】**
基于杠杆平衡原理。藏经阁地板由16块可活动木板组成，每块木板下方连接一个平衡秤系统。

**【具体结构】**
```
地板系统：
- 木板尺寸：60cm×60cm×5cm（柏木）
- 平衡秤：木制杠杆，支点可调，配重可调
- 触发阈值：±5kg重量变化
- 传动系统：麻绳滑轮组，连接至内部门闩

初始状态：
凶手提前调整平衡，使门闩处于“即将落下”的临界状态。

触发过程：
1. 尸体（70kg）放置于特定木板组合上
2. 重量改变打破平衡（ΔW=70kg）
3. 杠杆动作，通过滑轮组拉动门闩
4. 门闩落下，门从内部锁闭
5. 机关自动复位需要外部干预
```

**【工程验证】**
- 杠杆比：1:12（5kg力可拉动60kg门闩）
- 摩擦力：门闩滑轨涂桐油，摩擦系数μ=0.1
- 安全系数：木板承重100kg，安全系数2.0
- 灵敏度：通过调整配重实现±5kg触发阈值

**【道教文化契合】**
- 八卦阵朱砂线实际标记了地板活动范围边界
