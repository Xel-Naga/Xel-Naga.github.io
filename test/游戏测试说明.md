# 《悬观谜案：百年轮回》游戏测试说明

## 🎮 快速开始

### Windows用户（推荐）
1. **双击运行** `start_game_gbk.bat`
2. 自动打开浏览器访问：`http://localhost:8000`
3. 按 `Ctrl+C` 停止服务器

### 所有系统（命令行）
```bash
python start_server.py
```
然后打开浏览器访问：`http://localhost:8000`

- Python 3.6 或更高版本

### 浏览器要求
- 现代浏览器（Chrome 90+、Firefox 88+、Edge 90+）
- 支持ES6模块
- 建议开启开发者工具（F12）查看控制台

## 🐛 常见问题解决

### 问题1：Python命令找不到
**症状**：`'python' 不是内部或外部命令`
**解决**：
1. 重新安装Python，安装时勾选 **"Add Python to PATH"**
2. 或使用 `python3` 命令
3. 手动添加Python到PATH环境变量

### 问题2：批处理文件乱码
**症状**：中文字符显示为乱码
**解决**：使用 `start_game_gbk.bat` 替代 `start_game.bat`

### 问题3：端口8000被占用
**症状**：`[Errno 10048] Only one usage of each socket address`
**解决**：
1. 修改 `start_server.py` 中的 `PORT` 变量
2. 或关闭占用8000端口的程序
3. 运行命令检查端口：`netstat -ano | findstr :8000`

### 问题4：模块化版本CORS错误
**症状**：`Access to script ... has been blocked by CORS policy`
**解决**：必须通过HTTP服务器运行，**不要直接双击index.html**
```bash
python start_server.py
```

### 问题5：缺少requests库
**症状**：`ModuleNotFoundError: No module named 'requests'`
**解决**：
```bash
pip install requests
# 或
python -m pip install requests
```

## 📊 模块化架构优势

将4029行的单一大文件拆分为15个模块：

| 模块类型 | 数量 | 功能 |
|---------|------|------|
| 核心模块 | 3 | 游戏控制器、状态管理、事件系统 |
| 管理器模块 | 6 | 状态、决策、道具、任务、对话、地图 |
| 系统模块 | 5 | 交互、UI、移动、时间、数据加载 |
| 工具模块 | 2 | 常量定义、辅助函数 |

**优点**：
- 职责分离，每个模块200-400行
- 松耦合设计，通过事件系统通信
- 易于维护和扩展
- 便于单元测试和调试
