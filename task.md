### 🎯 技术实现要点

#### 1. 颜色标注交互系统
```javascript
// 7种颜色分类标准
const COLOR_CLASSES = {
  'interactable': '可互动对象（点击执行动作）',
  'important': '重要信息（关键剧情点）',
  'danger': '危险警告（可能影响理智值）',
  'clue': '线索标记（可收集到笔记）',
  'item': '道具相关（可获得/使用道具）',
  'npc': 'NPC相关（触发对话）',
  'location': '地点切换（前往新位置）'
};
```

#### 2. 道具系统数据结构
```json
{
  "id": "item_invitation",
  "name": "破旧邀请函",
  "type": "document",
  "rarity": "common",
  "description": "一张泛黄的邀请函，字迹有些模糊",
  "detailed": "邀请函内容：'诚邀参加青玄观百年修缮讨论会...'",
  "usable": true,
  "useEffect": "触发对话选项",
  "combineWith": ["item_magnifier"]
}
```

#### 3. 任务触发机制
- **位置触发**：到达特定位置时自动触发任务
- **道具触发**：获得/使用特定道具时触发任务
- **对话触发**：完成特定对话后触发任务
- **时间触发**：游戏内时间到达特定时刻触发

#### 4. 对话树系统
- 支持多层级分支对话
- NPC关系值影响对话选项可用性
- 对话选项可触发任务、获得道具、解锁位置
